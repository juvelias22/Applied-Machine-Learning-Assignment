Civilian Complaint Assessment using CatBoost,
K-Nearest Neighbor & Logistic Regression

CatBoost Model - Elias Cajal

1) Start by selecting the duration in months of the complant.
2) Check the corresponding checkbox if the member of service or the complainant is male.
3) Select the ethnicites of the complainant and the member of serves using their respective dropdown menus.
4) select one of the availabe FADO type allegation.

K-Nearest Neighbors - Shaun Tonna

1) Start by selecting the duration in months of the complant.
2) Select an allegation type by specifying the value representing that allegation. The values for the allegations are as follow:
0: Abuse of Authority      1: Discourtesy      2: Force      3: Offensive Language

Logistic Regression Model - Isaac Montebello

1) Start by selecting an ethnicity from the available options from the two dropdown menus, which refer to the complainant and the defendant ethniciteis.
2) After that, choose one type of allegation by checking the respective checkbox.
3) Finally, use the slider to change the duration of the allegation.
